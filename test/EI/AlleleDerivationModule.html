      4. Chemically induced (other)
      5. Endonuclease-mediated
      6. Gene trapped
      7. Not Applicable
      8. Not Specified
      9. Other
      10. QTL
      11. Radiation induced
      12. Spontaneous
      13. Targeted
      14. Transgenic
      15. Transposon induced

  4. Verify creating a derivation from mutant cell line is correct

    jeffc 02/03/2014

    1. Open the Scrum-dog EI
    2. Log in to the Scrum-dog EI
    3. Click on the Allele module button
    4. From the Edit menu, select Mutant cell line
    5. Enter Creator Achim Gossler, Parent cell line 2A, Cell Line Type Embryonic Stem Cell, Deriation Type Targeted, Vetor Names eFlpROSAbetageo, Vector Type enhancer trap, and Allele Symbol Pros1<tm1.2Grl> and click Add
    6. Go to the Allele Derivation module and search for the creator Achim Gossler
    7. Verify one of the results is for Achim Gossler Targeted Library 2A129S1/SvImj eFlpROSAbetageo.
