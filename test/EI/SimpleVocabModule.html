EI Simple Vocabulary module

  • User story 16, EI Allele Module supports collections(simple vocab piece)
  • User story 22, E5: EI Allele and Cell Line modules(simple vocab piece)

Acceptance Criteria

  • US16: Verify a Vocabulary Name called Allele Collection exists
  • US16: Verify the correct terms are listed in the Allele Collection
  • US16: Verify you can add a new term to the Allele Collection
  • US16: Verify you can change a term's name in the Allele Collection
  • US16: Verify you can delete a collection term that is not used
  • US16: Verify you can not delete a collection term that is being used
  • US22: Verify a Vocabulary Name called Allele Type exists
  • US22: Verify the correct terms are listed in the Allele Type
  • US22: Verify you can not add an Allele Type term
  • US22: Verify you can not delete an Allele Type term
  • US22: Verify you can not modify an Allele Type term
  • US22: Verify a Vocabulary Name called Allele Subtype exists
  • US22: Verify the correct terms are listed in the Allele Subtype
  • US22: Verify you can not add an Allele Subtype term
  • US22: Verify you can not delete an Allele Subtype term
  • US22: Verify you can not modify an Allele Subtype term
