EI Allele module

  • User story 16, EI Allele module supports collections
  • User story 22, E5: EI Allele and Cell Line modules
  • User story 85, E7: Load the IMSR feed file in MGI

Acceptance Criteria

  • US16: Verify a collection list exists
  • US16: Verify you can search by collection list
  • US16: Verify you can add a collection to an Allele
  • US16: Verify you can delete(set to not specified) a collection to an Allele
  • US16: Verify you can change(modify) the collection selection for an Allele
  • US22: Verify you can search by Generation type
  • US22: Verify you can search by Allele Attribute
  • US22: Verify you can change generation type to an Allele
  • US22: Verify you can add an allele attribute to an Allele
  • US22: Verify you can change an allele attribute of an Allele
  • US22: Verify you can add multiple allele attributes to an Allele
  • US85: Verify alleles using J:204812 all have the TRansmission status of Germline
