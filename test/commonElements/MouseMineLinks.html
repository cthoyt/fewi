Links to MouseMine within the MGI web Interface

  • User story 110, Links to MouseMine

Acceptance Criteria

  • All links to Mousemine display and link to the Mousemine product
  • The link to MPD from the More Resources pulldown uses the correct URL

Tests

Skip to Manual Tests | Automated Tests

Manual tests

  1. Verify the Search pulldown menu contains a working link to Mousemine

    jeffc 03/12/2013

    1. Open the MGI main page and hover over the Search pulldown menu
    2. Verify a link to MouseMine exists just after MouseBLAST
    3. Verify clicking on the Mousemine link takes you to the Mousemine main page
  2. Verify the All Searches page contains a link to MouseMine

    Jeffc 03/12/2013

    1. Open the MGI main page and hover over the Search pulldown menu
    2. Click on the All Search Tools link
    3. Verify a link to MouseMine exists in the Genes, Genome Features & Maps section just below the Mouse Genome Browser link
    4. Verify clicking on the Mousemine link takes you to the Mousemine main page
  3. Verify the Analysis Tools page contains a link to Mousemine

    jeffc 03/12/2013

    1. Open the MGI main page and click on the Analysis Tools link
    2. Verify a link to MouseMine exists in the Genome Analysis Tools section just below the Batch Query link
    3. Verify clicking on the Mousemine link takes you to the Mousemine main page
  4. Verify the More Resources pulldown list link to MPD uses the correct URL

    jeffc 03/13/2013

    1. Open the MGI main page and hover over the More Resources pulldown link
    2. Verify when you hover over the Mouse Phenome Database link the URL is phenome.jax.org
    3. Verify clicking on the MPD link takes you to the Mouse Phenome Dayabase main page
