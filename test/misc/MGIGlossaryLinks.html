Links to MGI Glossary within the MGI Web Interface

  • User story 52, Convert Glossary pages to FEWI

Acceptance Criteria

  • All links to MGI Glossary will use the structure http://<server name>/fewi/mgi/glossary/ or http://<server name>/fewi/mgi/glossary/<key>
  • The MGI Glossary main page will have the URL of http:///fewi/mgi/glossary/
  • All links to the glossary or a term will use the new fewi link
