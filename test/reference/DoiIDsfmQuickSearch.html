TR11313 - Quick Search returns Bad Link for DOI IDs

  • User story 127, Quick search returns bad link for DOI ids

Acceptance Criteria

  • All DOI ID searches from quick search return a reference link that works and is correct
  • EXCEPT: those IDs that contain a semi-colon will fail. Semi-colon and space are list delimiters.
