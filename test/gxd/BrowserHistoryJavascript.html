Acceptance Criteria

GXD query form and summary browser navigation intended Behavior

  • back button takes you back one event reliably
    • back button returning to initial query form should display the query form with default values
  • fwd button takes you forward one event reliably

Events are defined as

  • pagination change
  • tab change
  • query form submit

Query form submit behavior

  • submits query
  • display defaults to Results tab, page 1

Tab change behavior

  • display returns to default: to page 1 for the selected tab
