Marker Detail GXD Link Changes

  • User story 96, Change links to GXD from the Marker Detail page

Acceptance Criteria

  • Assays links from Expression section of marker detail are removed
  • Results links from Expression section of marker detail go to unique GXD Assay Results page
  • TS links from Expression section of marker detail go to unique GXD Assay Results page
  • Unique GXD Assay Results page has no modify search section
  • Unique GXD Assay Results page has the standard Marker heading ribbon
  • Unique GXD Assay Results page has You searched for.. section with text appropriate to the referring link
