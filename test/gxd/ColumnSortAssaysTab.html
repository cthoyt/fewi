Column Sort Assays Tab

  • User story 86, Column Sort Assays Tab

Acceptance Criteria

  • data are sortable by:
    • assay type
    • gene
    • reference
    (each has more layers)

Info

Default sort order for results set on the "Assays" tab is:

  • Gene symbol
    • Assay Type
      • Reference

All Assay Type sorting occurs in this order:

  1. Immunohistochemistry
  2. RNA in situ
  3. In situ reporter (knock in)
  4. Northern blot
  5. Western blot
  6. RT-PCR
  7. RNase protection
  8. Nuclease S1

Gene symbol is always sorted by MGI's "smart-alpha" algorithm (where embedded numbers are treated numerically, not ASCII-sorted).

Reference is always sorted alphabetically by the mini-citation text and disregards J# in the sort algorithm.
