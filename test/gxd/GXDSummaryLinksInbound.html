GXD Summary Links, Inbound

formerly: Marker Detail GXD Link Changes

  • MGI 5.10, User story 96, Change links to GXD from the Marker Detail page
  • MGI 5.14, User story 81, Change links to GXD Summary from the Antibody Detail and Probe Detail pages
  • MGI 5.20, User story 22, E2: Add links to tissue x stage results

Acceptance Criteria

  • sto22: add new link from marker detail expression ribbon
  • sto22: link text = "Matrix view"
  • sto22: link to expression results for that gene, opened to the tissue x stage matrix tab

Functional Requirements

  • "Assays" link from Expression ribbon of the Marker Detail Page does not appear
  • "Results" link from Expression ribbon of the Marker Detail Page goes to non-query GXD Assay Results page
  • "TS" link from Expression ribbon of the Marker Detail Page goes to non-query GXD Assay Results page
  • Non-query GXD Assay Results page has no "modify search" section
  • Non-query GXD Assay Results page has the standard "Object" heading ribbon (marker, reference, allele, or other object)
  • Non-query GXD Assay Results page has You searched for.. section with text appropriate to the referring link, when appropriate
  • "Results" tab is the default active tab except...
    • "Tissues X Stage Matrix" tab is the default active tab when following "Matrix View" link from Marker Detail page
