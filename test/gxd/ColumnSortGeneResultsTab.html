Genes Tab Column Sorting

  • User story 28, Build column sorting for Gene Summary

Acceptance Criteria

  • Icons appear for sorting in the column heading for the columns Gene and Chr
  • Sorting of the Gene column is using the default sort(smart alpha)
  • Sorting of the Chr column (3 columns together chr, genome location, cM) has secondary sort of Gene
  • When icon is clicked, all data in the results set is resorted by the values in the clicked column (no secondary sort allowed by user)

Tests

Manual tests

  1. Clicking "Gene" header performs multi-column sort:

    passed jeffc 07/19/2012

    1. Open the GXD query form
    2. Search for trp*
    3. Click the Genes tab
    4. Click "Gene" header
    5. Verify data are returned in the following order
    6. Gene Symbol (smart alpha)
  2. Clicking "Chr" header performs multi-column sort:

    jeffc 07/19/2012

    1. Open the GXD query form
    2. Search for brain in the Anatomical Structure field
    3. Click the Genes tab
    4. Click "Chr" header twice
    5. This has the Chr column sorted by reverse order of MT,UN,XY,Y,X,19-01
    6. Verify data are returned in the following order
      ChrGene Symbol
      smart Alphasmart alpha
    7. Find the genes with chromosome X and a location of Syntenic, these genes should be sorted by smart alpha
  3. Clicking "Chr" header a second time reverses the sort

    passed jeffc 07/19/2012

    1. Using the results from the last test, Click the "Chr" header again
    2. Verify the results are in reverse order for the Chr column with the order being 01-19,X,Y,XY,UN,MT
