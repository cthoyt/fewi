Acceptance Criteria

  • US14 - Standard MGI header displays and all pulldowns and hover text work correctly
  • US14 - Assay section displays new ribbon design with the items
    • reference, J number links to the proper Reference detail page
    • assay type
    • MGI accession ID (of the Assay)
    • gene symbol, Marker symbol links to the proper gene detail page
    • gene name
    • Modification date
  • US15 - (if reporter gene exists) Probe section displays reporter gene
    • US15 - (if reporter gene exists AND antibody data exist) also display antibody data, antibody name links to correct antibody detail page
    • US15 - (if reporter gene exists AND probe data exist) instead, also display probe data, probe name links to correct probe detail page
    • US15 - (if reporter gene exists and NO antibody and NO probe) instead, display text "direct detection"
  • US15 - (if probe) Probe section displays probe, probe preparation, and "visualized with" data (when these data exist)
  • US15 - (if probe) probe name links to correct probe detail page
  • US15 - (if antibody) Probe section displays antibody and "detection system" (when these data exist)
  • US15 - (if antibody) antibody name links to correct antibody detail page
  • US15 - display assay notes (if data exist)
  • US17 - (in situ)Table has columns of Structure, Level, Pattern, Image and Note
  • US17 - (in situ)Confirm if Pattern is "not specified" or "not applicable" the field is left blank.
  • US17 - (in situ)Results Table Data for the Image column displays the proper figure labels and links properly
  • US16 - (blot)Table has table headings of Sample Information, Bands, Other Sample Information. It should have sub-headings of Lane, Age, Structure, (band sizes), Amount, Genetic Background, Mutant Allele(s), Sex and Note
  • US16 - (blot)Table displays reference to lane notes, band notes, row notes, and age notes in the correct locations. (lane notes display in the note column, all other notes display as table footnotes)(Conditional notes cover in another test)
  • US16 - (blot)Table displays Lane 1, Lane 2, etc if the Label field of the EI is blank.
  • US16 - (blot)Table displays superscript correctly when used in a lane label
  • US16 - All Western blots should suppress the Amount column
  • US16 - (blot)Table remove any trailing zeros from gel row or band sizes. EG: 300.0 or 12.0
  • US16 - (blot)Table, if sample amount and rna type are both Not Specified display Not Specified in stead of showing Not Specified; Not Specified RNA
  • US16 - If any unused lane has a lane note in the database do not display it in the WI
  • US16 - Verify that Not Applicable is not suppressed in Band rows
  • US16 - Verify if a band row size field is null the heading is set to Size Not Specified no matter what row unit might be selected
  • US65 - page title is in the format symbol + assay type + data
  • US23 - each results section displays a short summary line for that specimen (with "more" and a turnstile in parentheses)(age is separated from Genotype by a semicolon)
  • US23 - If there is a specimen note, it is displayed below the [specimen data] line
  • US23 - When the "more" turnstile is opened, additional specimen details are displayed in table below the [specimen data] line [and above the specimen note]
  • US23 - hemizygous X- and Y-linked genotypes should display as "allele/X" or "allele/Y" (rather than current "allele/blank")
  • US23 - Add the text "Conditional mutant specimen" to the beginning of all specimen notes where genotype is conditional, if there is no other specimen note, the conditional text becomes the note
  • US18 - Image panes will be displayed above the Results table
  • US18 - Image and panel/figure labels link to their image detail page
  • US18 - Image panes should be displayed in alpha-numeric order when a specimen has more than one image
  • US18 - Multiple panes should wrap when hitting page width
  • US18 - Add a copyright icon to the pane label that links to the image detail page
  • US19 - Image panes will be displayed above the Results table
  • US19 - Image and panel/figure labels link to their image detail page
  • US19 - When a gel has an image reference but no jpg just display the Figure label
  • US19 - Add a copyright icon to the pane label that links to the image detail page
  • US22 - the stakeholders and the team have agreed on a set of rules for (re)sizing displayed image panes (see comments to a strawman set of such rules)
  • US22 - the rules are implemented
  • Sto54: MGI IDs in Specimen notes will link to correct associated page
  • Sto55: MGI IDs in Assay notes will link to correct associated page
