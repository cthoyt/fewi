GXD Query Form and Results

Functional Description

Planned Querying Functionality:

The user may search for expression data for genes by identifying the genes by nomenclature, vocabulary term annotations, or chromosomal location. The user also may search for expression data associated with particular anatomical structures or developmental stages, in specimens bearing mutations in particular genes, or by the type of expression assay performed. The user may specify whether they wish to restrict the results only to data where expression was detected, only to data where expression was assayed but was absent, or to include all data.

Planned Results Summary Functionality:

The user may access each of three views of the results set of data using "tab" affordances: Genes, Assays, or Assay Results. Each view contains a data table with particular columns, sorting and filtering requirements. Results will be presented in "paginated" subsets. The user may download the full results set in tab-delimited or Excel format. The user may forward the list of returned genes to the Batch Query tool.
