GXD Query Form and Results

Functional Description

Planned Querying Functionality:

Standard Query Form: The user may search for expression data for genes by identifying the genes by nomenclature, vocabulary term annotations, or chromosomal location. The user also may search for expression data associated with particular anatomical structures or developmental stages, in specimens bearing mutations in particular genes, or by the type of expression assay performed. The user may specify whether they wish to restrict the results only to data where expression was detected, only to data where expression was assayed but was absent, or to include all data.

Differential Query Form: The user may search for genes having positive expression results in structure "A" but not structure "B". The user may search for genes having positive expression results in some stage(s) but not (an)other stage(s). The user may search for genes having positive expression results in structure "A" at certain stage(s) but not structure "B" at (an)other stage(s). The query returns positive expression results for the selected structure and its descendant terms and/or stage(s) from the first part of the query. The query returns negative results for the selected structure and/or stage(s) from the second part of the query, but does not return negative results for ancestor or descendant terms.

Planned Results Summary Functionality:

The user may access each of four views of the results set of data using "tab" affordances: Genes, Assays, Assay Results, or Images. Each view contains a data table with particular columns, sorting and filtering requirements. Results will be presented in "paginated" subsets. The user may download the full results set in tab-delimited or Excel format. The user may forward the list of returned genes to the Batch Query tool.
