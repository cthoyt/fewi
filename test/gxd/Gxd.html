Functional Description Overview

Querying Functionality:

Standard Query Form: The user may search for expression data for genes by identifying the genes by nomenclature, vocabulary term annotations, or chromosomal location. The user also may search for expression data associated with particular anatomical structures or developmental stages, in specimens bearing mutations in particular genes, or by the type of expression assay performed. In querying by anatomical structure, the user must select an EMAPA term from the pick list provided (added 2014). The user may specify whether they wish to restrict the results only to data where expression was detected, only to data where expression was assayed but was absent, or to include all data.

.

Differential Query Form: The user may search for genes having positive expression results in structure "A" but not structure "B". The user may search for genes having positive expression results in some stage(s) but not (an)other stage(s). The user may search for genes having positive expression results in structure "A" at certain stage(s) but not structure "B" at (an)other stage(s). In querying by anatomical structure, the user must select an EMAPA term from the pick list provided (added 2014). The query returns genes having positive expression results for "A" or its descendants in selected stage(s) which have no positive expression results for "B" or any of its descendants in selected stage(s). For those genes, the query returns positive expression results for structure "A" and its descendant terms for selected stage(s). For the same genes, the query returns negative results for structure "B" for selected stage(s), but does not return negative results for ancestor or descendant terms. Results that are "ambigous" or "not specified" are not considered in the query are never returned. Only "wild type" specimens are included, where "wild type" is specially defined.

Results Summary:

The user may access each of six views of the results set of data using "tab" affordances: Genes, Assays, Assay Results, Images, Tissue X Stage matrix (added 2014), or Tissue X Gene matrix (added 2014). Each view contains a data table with particular columns, sorting and filtering requirements. Applying filters affects the returned data in all "tab" views. Results are presented in paginated subsets. The user may download the full results set in tab-delimited or Excel format. The user may forward the list of returned genes to the Batch Query tool.

Expression Assay Detail Page

The Assay Detail page includes a separate results table and relevant image panes for each Specimen in the Assay.
