Assay Detail - Fewi Inbound Links Tests

  • User story 21, GXD Assay Detail - fix inbound fewi links

Acceptance Criteria

    fix inbound fewi links: the links to assay detail pages should be in the format cardolan.jax.org/assay/MGI:[nnnnn]
  • from GXD query results
  • from image detail
  • from assay summary
  • from accession summary

Tests

Skip to Manual Tests | Automated Tests

Manual tests

  1. The Accession "query" functionality forwards to the new FeWI Assay Detail Page.

    klf 3/14/2013

    1. Enter the URL: cardolan.informatics.jax.org/accession/MGI:1859758
    2. Verify that the page rendered is NOT cardolan.informatics.jax.org/searches/expression.cgi?4552.
    3. Verify that the page rendered IS cardolan.informatics.jax.org/assay/MGI:1859758.
  2. The link from an Expression Image Detail page goes to the new FeWI Assay Detail Page.

    klf 3/14/2013

    1. Navigate to an Image Detail Page: test case
    2. In the Associated Assays ribbon, verify that the links to assay detail pages include http://cardolan.informatics.jax.org/assay/MGI:1859758 and http://cardolan.informatics.jax.org/assay/MGI:1859819
    3. Verify that following the links renders the correct pages.
  3. The link from an Expression Query Results Summary ("Standard" query form) goes to the new FeWI Assay Detail Page.

    klf 3/14/2013

    1. Navigate to the GXD Query Form and perform a query for "Pax1".
    2. On the Results tab, in the Result Details column, verify that the link for MGI:1334713 is in the form cardolan.informatics.jax.org/assay/MGI:1334713.
  4. The link from an Expression Summary for a Reference goes to the new FeWI Assay Detail Page.

    klf 3/14/2013

    1. Navigate to a References page: test case J:39896.
    2. Click the "Expression results" count: 128 (as of 3/14/2013).
    3. On the Results tab, in the Result Details column, verify that the link for MGI:2663420 is in the form cardolan.informatics.jax.org/assay/MGI:2663420.
    4. On the Assays tab, in the Assay Details column, verify that the link for MGI:2663420 is in the form cardolan.informatics.jax.org/assay/MGI:2663420.
  5. The link from an Expression Summary for a Reference + Marker goes to the new FeWI Assay Detail Page.

    klf 3/14/2013

    1. Navigate to a GXD Literature Index Detail page which links to detailed expression data for the assays: test case Pax6 + J:76822.
    2. Follow the link to "Detailed expression data for these assays:" 109 results (as of 3/14/2013).
    3. On the Results tab, in the Result Details column, verify that the link for MGI:5302971 is in the form cardolan.informatics.jax.org/assay/MGI:5302971.
  6. The link from an Expression Summary for a Marker goes to the new FeWI Assay Detail Page.

    klf 3/14/2013

    1. Navigate to a Marker Detail page: Pax6.
    2. Click the "Expression results" count: 1606 (as of 3/14/2013).
    3. On the Results tab, in the Result Details column, verify that the link for MGI:1859758 is in the form cardolan.informatics.jax.org/assay/MGI:1859758.
    4. On the Assays tab, in the Assay Details column, verify that the link for MGI:2178971 is in the form cardolan.informatics.jax.org/assay/MGI:2178971.
  7. The link from an Expression Summary for a Marker + Structure goes to the new FeWI Assay Detail Page.

    klf 3/14/2013

    1. Perform a Batch Query for the marker Pax6. Specify "Gene Expression" results in the output.
    2. In the "Assay Results" column, follow the link for 3:4-cell stage embryo: 25 (as of 3/14/2013).
    3. On the Results tab, in the Result Details column, verify that the link for MGI:5289542 is in the form cardolan.informatics.jax.org/assay/MGI:5289542.
    4. NOTE: This is the same summary page as the Marker Tissue summary so it behaves in the same way.
