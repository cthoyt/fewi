Assay Results Pagination Test Suite

  • User story 24, Build pagination for Assay Results data
  • User story 80, Change pagination default and picklist values

Acceptance Criteria

  • pagination controls exist and follow MGI standard behaviors
  • default sort
  • when clicked, displayed results set repopulates with appropriate values
  • changing results tab-view sets new summary to Page 1 of its data
  • US80: set default value to 100 rows per paginated "page"
  • US80: set picklist values to 50, 100, 250, and 500 (was 10, 25, 50, 100)
  • US80: applies to Genes and Assay Results tabs
  • US80: Assays tab is being set to these values during Sprint 6

Tests

Manual tests

  1. Pagination controls appear and conform to MGI standard

    Passed jeffc 08/10/2012

    1. Open the GXD query form
    2. Enter zfp* in the gene nomenclature field and press search
    3. Verify Pagination controls appear. The first and previous links should be inactive, the next and last links should be active
  2. Verify that paging forward one page works

    Passed jeffc 08/10/2012

    1. Open the GXD query form
    2. Enter zfp* in the gene nomenclature field and press search
    3. Click the next page control in the pagination
    4. Verify the second page of gene results should appear and the first and previous pagination controls should be active and the next and last pagination controls should be active
  3. Verify that paging backward one page works

    Passed jeffc 08/10/2012

    1. Open the GXD query form
    2. Enter zfp* in the gene nomenclature field and press search
    3. Click the next page control in the pagination
    4. Click the previous page control in the pagination
    5. The first page of gene results should appear and the first and previous pagination controls should be inactive and the next and last pagination controls should be active
  4. Verify that paging to the beginning of the results works

    Passed jeffc 08/10/2012

    1. Open the GXD query form
    2. Enter zfp* in the gene nomenclature field and press search
    3. Click the next page control in the pagination
    4. Click the first page control in the pagination
    5. The first page of gene results should appear and the first and previous pagination controls should be inactive and the next and last pagination controls should be active
  5. Verify that paging to the end of the results works

    Passed jeffc 08/10/2012

    1. Open the GXD query form
    2. Enter zfp* in the gene nomenclature field and press search
    3. Click the last page control in the pagination
    4. Verify the last page of gene results should appear and the first and previous pagination controls should be active and the next and last pagination controls should be inactive
  6. Verify that page size (i.e. results per page) control works on the Assay Results tab

    Passed jeffc 08/10/2012

    1. Open the GXD query form
    2. Enter zfp* in the gene nomenclature field and press search
    3. Change the page size to 50 and page back and forth through the Assay Results data
    4. Verify the results per page match the selected option, except for the last page where the results end
    5. Change the page size to 100 and page back and forth through the Assay Results data
    6. Verify the results per page match the selected option, except for the last page where the results end
    7. Change the page size to 250 and page back and forth through the Assay Results data
    8. Verify the results per page match the selected option, except for the last page where the results end
    9. Change the page size to 500 and page back and forth through the Assay Results data
    10. Verify the results per page match the selected option, except for the last page where the results end
  7. Verify that page size (i.e. results per page) control works on the Genes tab
