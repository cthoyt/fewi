Downloadable Reports Test Suite

  • User story 26, Build download options for assay results data

Acceptance Criteria

  • download controls exist (text and Excel format)
  • results set gets downloaded
  • downloaded report opens properly in target application

Manual tests Assay Results data

  1. Excel and Text download buttons appear on assay results output

    jeffc 8/10/2012

    1. Open GXD QF
    2. Submit a query
    3. Verify "excel" button appears in "Table Toolbar" above column header row and below output tabs (see requirements)
    4. Verify "text" button appears in "Table Toolbar" above column header row and below output tabs (see requirements)
  2. Excel button downloads results set as Excel file

    jeffc 8/10/2012

    1. Open GXD QF
    2. Enter Pax1 in the Nomenclature field and submit the query.
    3. Record number of results returned.
    4. Press the Excel button
    5. Verify a file downloads
    6. Verify downloaded file opens successfully in Excel by default
    7. Verify downloaded file has same number of output rows as number of results in query summary output(first row has column headers so total is plus 1)
  3. Text button downloads results set as tab-delimited file

    jeffc 8/10/2012

    1. Open GXD QF
    2. Enter Pax1 in the Nomenclature field and submit the query.
    3. Record number of results returned.
    4. Press the Text button
    5. Verify a file downloads
    6. Verify downloaded file opens successfully in a text editor
    7. Verify downloaded file has same number of output rows as number of results in query summary output
    8. Verify values in the first couple of rows are separated by tabs
  4. Excel file has correct output columns

    kstone 2013/05/20

    1. Open GXD QF
    2. Enter Pax1 in the Nomenclature field and submit the query.
    3. Press the Excel button
    4. Open the downloaded file and verify columns in order:
      1. Marker ID as MGI Gene ID (not in web results summary)
      2. Marker symbol as Gene Symbol
      3. Marker name as Gene Name (not in web results summary)
      4. Assay ID as MGI Assay ID
      5. Assay type
      6. Anatomical System
      7. Age (displaying data as we have it: E9.5 or whatever)
      8. Theiler Stage (displaying only numeric value as done in BioMart)
      9. Structure
      10. Detected (displays "Level")
      11. Images
      12. Mutant Allele(s) (display superscripted text inside angle brackets as we store in the DB)
      13. Reference (contains just the J:# ID)
      14. PubMed ID (not in web results summary)
      15. Citation (contains the short-form reference citation data displayed in the HTML output)

  6. Excel file has correct data

    klf 8/20/2012

    1. Verify Excel file is named "MGIgeneExpressionQuery_YYYYMMDD_HHmm.xls" where YYYY is the year, MM is the month, DD is the date, HH is the hour (military), mm is the minutes
    2. Verify Text file is named "MGIgeneExpressionQuery_YYYYMMDD_HHmm.txt" where YYYY is the year, MM is the month, DD is the date, HH is the hour (military), mm is the minutes
