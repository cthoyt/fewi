  2. Confirm the Acc ID search accepts MGI IDs and J numbers. (sto33)

    passed jeffc 11/05/2014

    1. Enter "J:28000" in the Accession ID field and click Search
    2. Verify the following error is returned: Error Found Reference(BIB_Refs) object with ID = J:28000, but no link URL has been defined.
    3. *note: this error is displayed because the Reference Detail page has not been created yet.
  3. Confirm that multiple MGI IDs and J numbers are not accepted input. (sto33)

    passed jeffc 11/05/2014

    1. Enter "MGI:97490, J:28000" in the Accession ID field and click Search
    2. Verify the following error is returned: Error No accession object found for ID = MGI:97490, J:28000
    3. *note: this error is because multiple IDs are not supported for the Accession ID search.
