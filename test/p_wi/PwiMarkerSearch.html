P-WI Marker Search Tests

  • User story 31, P-WI: Marker search by Nomen (Release 5.2?)

Acceptance Criteria

  • sto31: You can search by current symbol, Name, or mouse synonyms
  • sto31: Support the use of wildcard "%"
  • sto31: Withdrawn markers should be returned
  • sto31: Symbols are links to their Marker detail page
  • sto31: Existence of a link to view all marker records when results are more than 5000

  3. Confirm that symbols are links to their corresponding Marker detail page. (sto31)

    passed jeffc 11/06/2014

    1. Enter "pax%" in the nomenclature field and click Search
    2. Verify every symbol is a link

  4. Confirm clicking on "View all marker matches" display the complete set of markers. (sto31)

    passed jeffc 11/10/2014

    1. Enter "a%" in the nomenclature field and click Search
    2. Verify the first 5000 matching marker records are displayed
    3. Verify the existence of a link called "View all marker matches" exists just below the matching marker records heading.
    4. Confirm clicking on this link displays the complete set of matching marker records.
