Molecular Image Detail Page Test Suite

  • User story #55, Molecular Images: Molecular Image Detail page

Acceptance Criteria

  • clicking on a Molecular Image thumbnail takes you to it's Molecular Image Detail page
  • A Molecular Image Detail page contains:
    • Title
    • Full size image
    • Image caption
    • Copyright
    • Associated Allele, the symbol is linked to allele detail page
