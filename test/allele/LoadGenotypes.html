Genotype Loader Tests

  • User story 16 - Sanger Load: Genotypes

Acceptance Criteria

  • MGI input format for genotype loader defined
  • the sanger file is correctly parsed into MGI input format. Errors reported.
  • loader correctly reflects (in the db) the data in its load file
  • only creates genotypes
  • strains/alleles that it cannot find are reported

See also: LoadGenotypeCellLines and LoadPhenotypeAnnots.
