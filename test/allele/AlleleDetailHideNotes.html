Hide Molecular notes and Mutation origin text

  • User story 103, Hide Molecular note and Mutation Origin text behind turnstiles

Acceptance Criteria

  • Hide all information except Strain of Orgin behind a turnstile for the Mutation Origin ribbon
  • Hide all information except Allele Type and Mutation? behind a turnstile for the Mutation description ribbon
  • Mutation note does not use a turnstile icon if it's size is 100 characters or less

Requirements & Tests

Manual tests

  1. Allele detail page with turnstile icons for multiple sections

    jeffc 01/03/2013

    • Navigate to the Gene Trap Allele Detail Page for Arrdc3Gt(CSE151)Byg
    • In the Mutation Description section, confirm there are turnstile icons for Mutation Notes, Sequence Tags, and Genome Context
    • Verify clicking the turnstile icon for Mutation Notes displays the complete notes.
  2. Spontaneous Allele detail page display of a mutation note less than 100 characters

    jeffc 01/03/2013

    • Navigate to the QTL variant Detail Page for kitSsm
    • In the Mutation Description section, confirm no turnstile icon exists
    • Verify Mutation notes display without a turnstile icon since it is under 100 characters long
  3. Transgene detail page with Mutation notes just longer than 100 characters

    jeffc 01/03/2013

    • Navigate to the Transgene Detail Page for Tg(Ella-cre)C5379Lmgd
    • In the Mutation Description section, Confirm the Mutation notes has a turnstile icon
    • Verify clicking the turnstile icon for Mutation Notes displays the complete notes. In this case the notes should be just more than 100 characters long
  4. Transposon induced Allele detail page Mutation notes well over 100 characters

    jeffc 01/03/2013

    • Navigate to the Transposon induced allele Detail Page for Car12Tn(sb-T2/GT2/tTa)4563.1Dla
    • In the Mutation Description section, Confirm the Mutation notes has a turnstile icon
    • Verify clicking the turnstile icon for Mutation Notes displays the complete notes. These notes should be several lines long
