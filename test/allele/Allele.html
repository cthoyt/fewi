Allele Test Suites

Includes Loads, Data Migrations, WI Detail Page, Queries, and Summary Pages

Functional Description

For recombinase alleles (alleles which possess a Driver Note), the Recombinase Activity ribbon incorporates:

  • Display is pulled from the FEWI
  • Recombinase activity in all Anatomical Systems and Structures is summarized in an overview grid, similar to the Phenotype annotations grid
  • Users may submit observations about the recombinase allele which may be entered by curators and displayed here.

For the Allele Detail Page, the Phenotype Summary grid incorporates:

  • Pulling the display from the FEWI
  • Displaying sex-specific (male/female) annotations
  • Displaying "Normal" annotations for any MP term
  • Associating mutant cell lines for an allele with the genotype which carries the MP annotation
  • Displaying MP annotations loaded automatically from WTSI (Sanger) and Europhenome High-Throughput Phenotyping projects
  • Modifying the page such that Phenotype-by-Genotype text is contained only in pop-up pages, one per Genotype

Allele Test Suites

Some test suites have indicators for Build Sprint regression testing in the Notes column. A test may be marked High priority when 2 conditions are true: (1) There is a likelihood that the code could break or some error could occur. (2) It would be very important to fix if it did. If only one of those conditions is true, the test is marked Low priority. Otherwise, no priority is indicated.
