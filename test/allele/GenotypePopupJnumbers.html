Genotype Popup Page Tests

  • User story 154, WI: Fewer J numbers for notes

Acceptance Criteria

  • if a note shows the male/female glyph or "N", show the J#, otherwise
  • if there is >1 J# for the MP term, show the J#, otherwise don't show the J#
  • All J# should still be listed following the MP term. This tweak only affects the notes.
