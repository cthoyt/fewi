Mutation Description Ribbon on Allele Detail Page

  • MGI 5.18 User story 24, E3: Display allele subtypes on allele detail and cre specificity pages
  • MGI 5.18 User story 54, Molecular Images: show thumbnail on Allele Detail page

Acceptance Criteria

  • sto24 - an allele's subtypes are displayed on the detail page
  • sto24 - if no subtypes, nothing displayed (!)
  • sto54 - is single molecular image for an allele display on allele detail page
  • sto54 - if more than one molecular image for an allele display first image only
  • sto54 - display molecular image in the mutation description ribbon floated to the right
