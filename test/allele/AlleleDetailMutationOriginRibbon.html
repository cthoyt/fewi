Mutation Origin Ribbon on Allele Detail Page

  • (MGI 5.18) User story 17, E3: Display collection on allele detail and cre specificity pages
  • (MGI 5.18) User story 85, E7: Load the IMSR feed file in MGI

Acceptance Criteria

  • an allele's collection is displayed on the detail page(sto17)
  • a cre allele's collection is displayed on the cre specificity page(sto17)
  • if no collection, nothing displayed (!)(sto17)
  • Alleles with transmission type of cell line but have a strain count for IMSR have their transmission type changed to Germline
