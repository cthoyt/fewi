Add Used-FC Reference

  • User story 161, Add J:# with Reference Type = Used-FC

Acceptance Criteria

  • There should be no Used-FC reference for our J:#s for an allele unless that allele also has genotypes with HTMP annotations.
  • An allele with annotations from a single provider should have Used-FC ref only from that provider.
  • An allele with annotations from both providers should have Used-FC ref from both providers.
