HDP Grid Summary Tab Tests

  • User story 137, Add links to gene. allele and disease detail pages
  • User story 140, Grid: Basic Filtering

Acceptance Criteria

  • US137: Mouse Gene symbols link to MGI gene detail pages
  • US137: Human Gene symbols link to MGI homology pages
  • US137: Human Gene Symbols that have no associated MGI homology page will have no link
  • US140: Existence of check boxes for both rows and columns, Go button, reset button, and Invert button
  • US140: Only checked rows and columns with data are returned
  • US140: The Reset button should remove all checks for both rows and columns
  • US140: The Invert button should flip the results by returning all rows and columns that do not have a check
  • US140: Results returned on the Grid Tab should be reflected on the Genes and Disease Tabs too
  • US149: Filtering should be applied across multiple pages
