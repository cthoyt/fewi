HDP Grid Summary Tab Tests

  • User story 137, Add links to gene. allele and disease detail pages
  • User story 140, Grid: Basic Filtering
  • User story 189, Grid: add hover text for grid cells

Acceptance Criteria

  • US137: Mouse Gene symbols link to MGI gene detail pages
  • US137: Human Gene symbols link to MGI homology pages
  • US137: Human Gene Symbols that have no associated MGI homology page will have no link
  • US140: Existence of check boxes for both rows and columns, Submit Filters button, Remove Filters button
  • US140: Only checked rows and columns with data are returned
  • US140: The Remove Filters button should remove all checks for both rows and columns
  • US140: Results returned on the Grid Tab should be reflected on the Genes and Disease Tabs too
  • US140: Filtering should be applied across multiple pages
  • US189: Hovering over a phenotype checkmark displays the gene(s) and System involved
  • US189: Hovering over a disease checkmark displays the gene(s) and Disease term involved
