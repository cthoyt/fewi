HDP: Add a Feedback Button to the Human Disease Query page

  • User Story 108, Add a "Your input Welcome" button to the Human Disease Query page

Acceptance Criteria

  • US108: Add a "Provide Feedback" button to the Human Disease Query page
  • US108: Confirm the button displays hover text when you hover over it
  • US108: Confirm clicking the button displays a feedback form
  • US108: Confirm all fields on the feedback form are editable
  • US108: Confirm the Reset button clears all entered data
  • US108: Confirm the Cancel button takes you back to the Human Disease query page
  • US108: Confirm after submitting the form you get directed to a confirmation page
  • US108: Confirm after submitting the form it is received by user support
