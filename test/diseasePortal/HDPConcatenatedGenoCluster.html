Concatenated Genotype Cluster page

  • User story 145, Concatenated Genotype Cluster page

Acceptance Criteria

  • each genotype cluster line in a popup links to a "new" page (really, a variant of an existing page)
  • this page concatenates the genotype details of the genotypes in that cluster
  • same format as the concatenated genotype popup off the allele detail
  • numbering (e.g., ht1, ht2, ...)isn't really needed
