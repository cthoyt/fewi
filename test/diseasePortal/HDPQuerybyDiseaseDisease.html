HDP: Querying by Disease, Disease output

  • User story 17, E2: Query by disease, debugging display of diseases
  • User story 58, E2: Stop words for Disease/Pheno query

Acceptance Criteria

  • user can enter OMIM ids, terms, synonyms(sto17)
  • diseases tab shows matching diseases (debugging output)(sto17)
  • genes displayed on gene summary tab(sto17)
  • Stop words get ignored and display all data when used by themselves(sto58)

Definitions

For current definitions of genotypes included or excluded from the various displays, see the HDPQuerybyDiseaseGene test suite.
