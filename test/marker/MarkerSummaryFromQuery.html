Logic Summary

Querying in Text Fields

Description of logic used in each of the text field inputs when a query is submitted:

Nomenclature

  • Performs a tokenized query against Solr index.
  • Initial and trailing wildcard symbols (asterisk) are supported; "internal" asterisks are treated literally.
  • Tokens are combined with Boolean AND.
  • Does not include search against allele synonyms.
  • Attempts to recreate, as much as possible, previous code's default sorting if Nomen field is submitted (except now using Solr field boosts):
    MRK_label table, April 2014, additional code sorted "exact" matches ahead of matches contained within text strings.
    labeltype labeltypename organism name priority
    MS current symbol mouse 1
    MN current name mouse 2
    AS allele symbol mouse 3
    AN allele name mouse 4
    MS old symbol mouse 5
    MN old name mouse 6
    MY synonym mouse 7
    MY human synonym human 8
    MY rat synonym rat 9
    MY related synonym 10
    OS human symbol human 11
    ON human name human 12
    OS rat symbol rat 13
    OS rhesus macaque symbol rhesus macaque 14
    OS cattle symbol cattle 14
    OS dog symbol dog, domestic 14
    OS zebrafish symbol zebrafish 14
    OS chicken symbol chicken 14
    OS chimpanzee symbol chimpanzee 14

GO and Protein Domain (InterPro)

  • Performs a tokenized query against Solr index.
  • No wildcards. Uses MGI's standard "contains text string" approach (e.g., query for rough also returns matches for through, query for ear would also return hear and heart).
  • Boolean support is desired in the future.

Pheno/Disease

  • Performs a tokenized query against Solr index.
  • Initial and trailing wildcard symbols (asterisk) are supported.
  • Returns only genes having annotated alleles in One Marker Genotypes (OMG) or "Simple" genotypes: 2-marker Conditional genotypes and 2-marker genotypes involving a Transgenic Reporter.
    • EXCLUDES markers for annotated alleles present only in Complex genotypes (unlike the Allele Query): genotypes having 3 or more markers.
  • Excludes Recombinase markers in Conditional genotypes.
  • Excludes Transgenic Reporter markers.
