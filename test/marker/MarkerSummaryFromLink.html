Logic Summary

Some marker summaries are generated by following a link from another MGI page. These will now use the FEWI technology stack.

  • References page previously used marker_report_by_reference.cgi
  • From quick search when you query an interpro term/id used marker_report.cgi
  • Special link on SNP summary when querying by genomic location range used marker_report.cgi
