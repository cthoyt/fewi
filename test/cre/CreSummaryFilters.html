Cre Summary Filtering Tests

Acceptance Criteria

  • sto51: cre summary supports filtering by the values in the detected-in system column
  • sto51: using MGI standard filtering LAF
  • sto84: cre summary supports filtering by the values in the not-detected-in system column
  • sto84: using MGI standard filtering LAF
  • sto52: cre summary supports filtering by the values in the inducible column
  • sto52: using MGI standard filtering LAF
