CRE Project Work - GXD Cre Snp Project

Functional Description

For the Cre Pages, we plan to modify the Cre activity query, the display of the summary page, and add an overview summary of all recombinase activity for an allele:

  • Allow more detailed querying for recombinase activity by using the AD structures rather than Anatomical Systems
  • Simplify display of query summaries
  • Create an overview summary of all activity data for each recombinase allele
  • Clean up links to external image sources from the Recombinase Specificy (detail) Page

Test Suites

Some test suites have indicators for Build Sprint regression testing in the Notes column. A test may be marked High priority when 2 conditions are true: (1) There is a likelihood that the code could break or some error could occur. (2) It would be very important to fix if it did. If only one of those conditions is true, the test is marked Low priority. Otherwise, no priority is indicated.
