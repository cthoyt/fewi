Manual tests

  1. Regress New notes field appears on GXDImage module of the EI.

    klf 6/6/13

    1. Open GXDImage Module of the EI
    2. Verify "External Link Notes" button appears and opens a Notes Dialog window.
  2. Regress Curators may enter new \Link(URL|display text|) MGI markup for arbitrary hyperlinks in this field. The \Link(...) markup is stored in the database.

    klf 6/6/13

    1. Log in to the EI with Update privileges.
    2. Open the GXDImage Module and search for images associated with J:184579.
    3. Select the Full Size image for "Calb2 Spinal cord (wh matter) P7", ID MGI:5429064.
    4. Open the External Link Notes and enter "\Link(http://connectivity.brain-map.org/transgenic/search?page_num=0&page_size=38&no_paging=false&exact_match=false&search_term=Calb2-IRES-cre&search_type=line|AIBS Data|)" without quotation marks.
    5. Click "Save Changes" to close the Notes Dialog window.
    6. Click "Modify" button in the GXDImage Module to save the note associated with that image.
    7. To verify data were saved: close then reopen the GXDImage Module.
    8. Open the External Link Notes and type "%Link%" without quotation marks.
    9. Click "Save Changes" to close the Notes Dialog window.
    10. Click "Search" button to search for Image records.
    11. Verify that the Full Size image for "Calb2 Spinal cord (wh matter) P7", ID MGI:5429064 is returned.
    12. Open the External Link Notes and verify the \Link(...) text appears.
